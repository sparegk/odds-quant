# Contributing

Create focused branches and small commits. Run backend tests, Ruff, mypy, and `python -m alembic check` before opening a pull request. Deployment changes must also pass the static deployment tests and backend image build in CI. Quantitative changes must document assumptions and include deterministic tests, especially for time cutoffs, settlement, calibration, and odds conversion.

Do not contribute unauthorized scrapers, bookmaker credentials, proprietary datasets, fabricated results, automated bet placement, or features encouraging loss chasing.
